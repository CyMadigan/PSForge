# Powershell Module Dependencies
- Plaster

# Installation

1. Clone this repository
2. Cd to the local clone and run `.\paket install` to get dependencies
3. Run `Import-Module .\PowershellDSCWorkflow.psm1`

# Available commands
- `New-DSCModule`
- `New-DSCResource`
- `Package-DSCModule`
- `Test-DSCModule`
